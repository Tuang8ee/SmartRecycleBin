Đây là git giúp việc gửi file và chia sẻ trong nhóm được dễ dàng hơn. 
Git này phụ vụ cho dự án thùng rác thông minh. Mọi người vào nhớ chú ý sắp xếp theo thứ tự thư mục nha.
- Thư mục Report: Lưu lại file báo cáo, các vấn đề liên quan đến báo cáo.
- Thư mục Document: Liên quan đến các tài liệu, doc nếu có.
- Thư mục Project: Sẽ lưu các vấn đề liên quan đến dự án như mạch, thiết kế cơ khí, hay code. Bên trong có các thư mục con:
  + Code
  + Altium
  + CAD
